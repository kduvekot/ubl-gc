## Summary

- **Total releases:** 35
- **Date range:** 2006-01-19 to 2025-12-03
- **All dates verified** from OASIS documentation
- **Use ISO format** (YYYY-MM-DD) for git commit timestamps

## Notes

- Dates fetched using: `curl -s -A "Mozilla/5.0" https://docs.oasis-open.org/ubl/{stage}-UBL-{version}/`
- For some UBL 2.0 releases without HTML files, dates inferred from directory timestamps
- These dates will be used to set `GIT_AUTHOR_DATE` and `GIT_COMMITTER_DATE` for historically accurate git commits
