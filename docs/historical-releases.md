## Notes

- **OS (Official Standard):** Final released version
- **CS (Committee Specification):** Pre-release specification that has completed committee review
- **CSD (Committee Specification Draft):** Draft version under development
- **CSPRD (Committee Specification Proposed Recommendation Draft):** Draft approaching recommendation stage
- **COS (Candidate OASIS Standard):** Candidate for standardization
- **PRD (Proposed Recommendation Draft):** Early proposal stage

Each directory contains:
- HTML documentation
- PDF specification
- XML schemas (XSD)
- Semantic models (XML)
- Code lists
- Alternative representations (JSON, ASN.1 for some versions)
